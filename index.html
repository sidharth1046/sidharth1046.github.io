WORDLE

Wordle is a web-based word game created and developed by Welsh software engineer Josh Wardle. The game is simple: You have six chances to guess the day's secret five-letter word. Type in a word as a guess, and the game tells you which letters are or aren't in the word. The aim is to figure out the secret word with the fewest guesses.
